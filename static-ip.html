Static IP

تثبيت IP السيرفر

السيرفر لازم يفضل بنفس العنوان عشان SSH وCockpit وCoolify وWolow يفضلوا شغالين بدون تغييرات.

Static من Ubuntu

تضبط IP داخل السيرفر بملف Netplan. مناسب لما عايز النظام نفسه ماسك العنوان.

DHCP Reservation من الراوتر

تسيب Ubuntu على DHCP والراوتر يحجز نفس IP للـ MAC Address. أسهل وأأمن في بعض الشركات.

اختار IP صح

متختارش IP مستخدم من جهاز تاني. الأفضل يكون خارج DHCP Range أو تعمل Reservation من الراوتر.

1) اعرف اسم كارت الشبكة والـ Gateway

bash
ip route get 8.8.8.8
ip a

2) افتح ملف Netplan

bash
ls /etc/netplan/
sudo cp /etc/netplan/*.yaml /etc/netplan/backup.yaml
sudo nano /etc/netplan/00-installer-config.yaml

3) مثال إعداد Static IP

yaml
network:
  version: 2
  ethernets:
    eno1:
      dhcp4: false
      addresses:
        - 192.168.1.18/24
      routes:
        - to: default
          via: 192.168.1.1
      nameservers:
        addresses:
          - 1.1.1.1
          - 8.8.8.8
بدّل eno1 لو اسم الكارت مختلف

اسم الكارت ممكن يكون eno1 أو eth0 أو enp2s0. استخدم الاسم اللي ظهر معك في ip route get 8.8.8.8.

4) جرّب ثم طبّق

bash
sudo netplan try
sudo netplan apply
ip a
ip route
ping -c 4 8.8.8.8
ping -c 4 google.com
لو أنت داخل SSH

استخدم sudo netplan try أولاً. لو الشبكة قطعت، النظام يرجع تلقائي بعد ثواني بدل ما تقفل على نفسك الاتصال.

5) لو حصل خطأ ارجع للنسخة القديمة

bash
sudo cp /etc/netplan/backup.yaml /etc/netplan/00-installer-config.yaml
sudo netplan apply
