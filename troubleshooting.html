حل المشاكل الشائعة

صفحة مرجعية سريعة لما حاجة تقف: شبكة، SSH، Cockpit، Coolify، WOL، أو Firewall.

السيرفر مش ظاهر على الشبكة

bash
ip a
ip route
ping -c 4 192.168.1.1
ping -c 4 8.8.8.8
ping -c 4 google.com

SSH مش بيفتح

bash
sudo systemctl status ssh
sudo ss -lntup | grep :22
sudo ufw status numbered
hostname -I

Cockpit مش بيفتح

bash
sudo systemctl status cockpit.socket
sudo ss -lntup | grep 9090
sudo ufw allow 9090/tcp
sudo ufw reload

Coolify مش بيفتح

bash
docker ps --filter "name=coolify"
docker logs --tail=100 coolify
sudo ss -tulpn | grep :8000
sudo ufw status

WOL مش بيشغل الجهاز

  • تأكد LAN cable متوصل.
  • تأكد Wake-on-LAN Enabled في BIOS.
  • تأكد Network PXE Boot Disabled.
  • تأكد service شغال قبل إطفاء الجهاز.
  • استخدم MAC الصحيح لكارت LAN.
bash
sudo ethtool eno1 | grep Wake
systemctl status enable-wol.service
ip link show eno1

UFW قفل عليك الاتصال

لو فقدت SSH، ادخل بشاشة وكيبورد ونفذ:

bash
sudo ufw status numbered
sudo ufw allow 22/tcp
sudo ufw reload
# لو الدنيا بايظة خالص
sudo ufw disable
